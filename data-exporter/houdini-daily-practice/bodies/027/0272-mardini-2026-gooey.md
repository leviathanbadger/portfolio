### Mardini 2026 - Gooey

Today is day 9 of [Mardini 2026][mardini-2026]. The prompt for today is "Gooey".

It's a tar pit. With a *lot* of effort spent on the surroundings, not just the pit.
I was supposed to work on two practice projects today to get ahead on Mardini, but
I guess I'm only doing this one.

New stuff:

- Point Relax SOP
- Ripple Solver SOP -- yes, I've used this once before, but the result was awful.
  Now I'm using it again.
- Remesh Bubbles SOP
- Vellum Solver SOP with combined wire and cloth sims via stitch constraints. Also
  with vellum objects copied + transformed using Vellum Pack and Vellum Unpack SOPs
  so I don't have to set up all of the leaves manually.
- More practice with component builder, point instancer, AOVs, edits in post via
  slap comp, etc.
- More practice with subsurface scattering.
- Some practice with 'thin film' material properties for the tar material.

Note: HDRI is from GreyscaleGorilla.

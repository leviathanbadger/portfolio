### Mardini 2026 - Grow

Today is day 10 of [Mardini 2026][mardini-2026]. The prompt for today is "Grow".

I wasted a bunch of time attempting to get a simulated growing tree, with mixed
results -- being generous -- so eventually I decided to use one of the built-in
tree tools and do a Vellum sim inflating it.

New stuff:

- Practice with Solver SOP for a growing tree solver, even if I ended up throwing
  it away due to issues compounded by a lack of time.
- Vellum Balloon - combination of Vellum Cloth and Vellum Pressure constraints.
- Time Shift to reverse time.
- Python SOP to run python code for more powerful interaction with the geometry.
  (In this case blending a point attribute over a sequence of frames.)
- More practice with subsurface scattering. Also, blending between two materials
  driven by a parameter.

HDRI is from GreyscaleGorilla.
